# customer_behavior_analysis1

## Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI. The goal is to explore purchasing patterns, understand customer preferences, and generate insights that help businesses make data-driven decisions. The project includes data cleaning, exploratory data analysis (EDA), SQL queries, and an interactive Power BI dashboard.

## Dataset

The dataset contains customer shopping information with attributes such as:

* Customer Age
* Gender
* Product Category
* Purchase Amount
* Payment Method
* Review Rating
* Purchase Frequency

The dataset contains approximately **3900 customer records** used to analyze shopping trends and customer behavior.

## Tools & Technologies

* Python (Pandas) – Data cleaning and analysis
* SQL (PostgreSQL/MySQL/SQL Server) – Data querying and extraction
* Power BI – Data visualization and dashboard creation
* Jupyter Notebook – Data analysis environment
* Gamma – Report and presentation creation

## Steps Performed

### Data Loading

The dataset was imported into Python using Pandas for analysis.

### Data Cleaning

* Checked for missing values
* Handled null values
* Standardized column names
* Verified data types

### Exploratory Data Analysis (EDA)

Performed analysis to understand:

* Customer spending patterns
* Category-wise purchases
* Payment method usage
* Customer rating trends

### SQL Queries

SQL was used to extract insights such as:

* Total purchases by category
* Average purchase amount
* Most used payment methods
* Customer demographic analysis

### Data Visualization

The cleaned dataset was imported into Power BI to build an interactive dashboard showing customer behavior insights.

## Dashboard

The Power BI dashboard includes:

* Purchase amount by product category
* Payment method distribution
* Customer age group analysis
* Review rating trends
* Customer purchase frequency

These visualizations help identify shopping trends and customer preferences.

## Results / Insights

Key insights from the analysis:

* Certain product categories show higher purchase frequency
* Some payment methods are used more frequently by customers
* Customer age groups display different purchasing patterns
* Review ratings provide insights into customer satisfaction

These insights help businesses improve marketing strategies and product offerings.

## How to Run the Project

1. Download the dataset.
2. Open the Jupyter Notebook file.
3. Install required libraries:
   pip install pandas numpy matplotlib seaborn
4. Run the notebook cells to perform analysis.
5. Open the Power BI (.pbix) file to view the dashboard.
